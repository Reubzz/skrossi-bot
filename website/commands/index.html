Commands

!rank Cooldown: 5 secs

Returns an Image that Shows you the Current Rank, XP-Progress and your Position in the leaderboard you have in the server.

!leaderboard Cooldown: 5 secs

Shows the Leaderboard for the server.

!tictactoe

Classic Tic-Tac-Toe Game using Discord Buttons. (2 Player Game)


Usage:

!tictactoe @user

!rps

Classic Rock-Paper-Scissors Game using Discord Buttons. (2 Player Game)


Usage:

!rps @<user>
